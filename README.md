# Amazon Clone - Backend API

A complete backend API for Amazon Clone built with Node.js, Express, and MongoDB.

## 🚀 Features

- User Authentication (Register/Login with JWT)
- Product Management (CRUD operations)
- Admin Dashboard (User & Product management)
- RESTful API with proper error handling
- MongoDB Integration for data persistence
- CORS Enabled for frontend integration

## 🛠 Quick Setup

1. **Clone repository**
   ```bash
   git clone https://github.com/harshitsagar/Amazon-Clone-Backend.git

2. **Install dependencies**
   ```bash
   npm install

3. **Environment Setup (.env)**
   ```bash
    MONGODB_URI=your_mongodb_connection_string
    JWT_SECRET=your_jwt_secret_key
    PORT=3002

4. **Run the Server**
   ```bash
    # Development
    npm run dev

    # Production
    npm start
